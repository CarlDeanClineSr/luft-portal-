# LUFT Results Directory

This folder contains all analysis outputs from the LUFT engine.

## Key Subfolders
- `test_001/` — Rebound test results
- `fft_sideband/` — FFT sideband detection (graviton AM signature search)
- `oulu_chi/` — Oulu cosmic ray χ fits
- `usgs_chi/` — USGS earthquake χ fits
- `magnetometer_chi/` — Magnetometer χ fits (USGS/INTERMAGNET Earth's magnetosphere data)
- `maven_chi/` — MAVEN Mars plasma χ fits
- `multi_science/` — Multi-source (FAST, VLA, CERN, etc.) χ fits

Check here for the latest χ cap confirmation across domains.
